Aim:

Create a centralized asset and resource management system that enables organizations to efficiently track, allocate, maintain, and manage physical assets and shared resources throughout their lifecycle. The platform aims to eliminate manual processes, prevent resource conflicts, streamline approvals, and improve asset visibility and utilization through a secure ERP-based solution.

Tools and Technology Use:

Frontend:
- React
- React Router (react-router-dom)
- Redux Toolkit (@reduxjs/toolkit)
- React Redux (react-redux)
- Vite (dev server/build)
- Tailwind CSS

Backend:

- Node.js
- Express.js

Database:

- Mongoose
- MongoDB
